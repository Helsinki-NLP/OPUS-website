আল্লাহই আমার জন্য যথেষ্ট, তিনি ছাড়া আর কোনো হক্ব ইলাহ নেই। আমি তাঁর উপরই ভরসা করি। আর তিনি মহান আরশের রব্ব।
{ يعلمون ظاهراً من الحياة الدنيا } أي معايشها من التجارة والزراعة والبناء والغرس وغير ذلك { وهم عن الآخرة هم غافلون } إعادة هم تأكيد.
তারা পার্থিব জীবনের বাহ্যিক দিক জানে, আর তারা পরকালের খবর রাখে না।
إن الذين آمنوا بالله ورسوله وعملوا الصالحات التي أُمروا بها، أولئك لهم نعيم مقيم في الجنات.
যারা ঈমান আনে আর সৎকাজ করে তাদের জন্য আছে নি‘য়ামাতে ভরা জান্নাত।
{ إنا لما طغا الماء } علا فوق كل شيء من الجبال وغيرها زمن الطوفان { حملناكم } يعني آباءكم إذ أنتم في أصلابهم { في الجارية } السفينة التي عملها نوح ونجا هو ومن كان معه فيها وغرق الاخرون.
এবং পৃথিবী ও পর্বতমালা উত্তোলিত হবে ও চুর্ণ-বিচুর্ণ করে দেয়া হবে,
الذين يؤدون الصلاة كاملة في أوقاتها ويؤتون الزكاة المفروضة عليهم لمستحقيها، وهم بالبعث والجزاء في الدار الآخرة يوقنون.
যারা নামায প্রতিষ্ঠা করে, যাকাত দেয় আর তারা আখেরাতের প্রতি দৃঢ় বিশ্বাস রাখে।
آية سجدہ (سجدة):
হে বস্ত্রাবৃত!
{ اهدنا الصراط المستقيم } أي أرشدنا إليه، ويبدَل منه:
আমাদেরকে সরল পথ দেখাও,
{ ليجزي } فيها { الذين آمنوا وعملوا الصالحات أولئك لهم مغفرة ورزق كريم } حسن في الجنة .
যাতে তিনি প্রতিদান দিতে পারেন তাদেরকে যারা ঈমান আনে ও সৎকাজ করে, এদেরই জন্য আছে ক্ষমা ও সম্মানজনক রিযক।
{ وعد الله الذين آمنوا وعملوا الصالحات } وعدا حسنا { لهم مغفرة وأجر عظيم } هو الجنة .
যারা ঈমান আনে ও সৎকাজ করে তাদের প্রতি আল্লাহ ওয়া‘দা করেছেন যে, তাদের জন্য রয়েছে ক্ষমা ও মহা পুরস্কার।
