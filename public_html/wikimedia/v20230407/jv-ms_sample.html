Windows Essentials (biyèné Windows Live Esensial lan Windows Live Installer) paket lunak, free kanggo kedhaftar Windows Genuine dening Microsoft kang diarahake kanggo kurban penak surel, olahpesen cepet, nuduhake foto, publishing blog, layanan keamanan lan liya-liyane sing klebu ing Windows Live.
Windows Essentials (sebelumnya Windows Live Essentials dan Windows Live Installer) adalah sebuah pakej perisian percuma untuk pengguna Windows Genuine oleh Microsoft yang ditujukan untuk menawarkan kemudahan e-mel, pesanan segera, perkongsian foto, penerbitan blog, perkhidmatan keselamatan dan lain-lain yang termasuk dalam Windows Live.
Iki paket intended kanggo kedhaftar saka Internet.
Pakej ini ditujukan untuk para pengguna Internet.
Windows Live Essentials wis integrasi karo Windows 7 nanging ora dibungkuus bebarengan ing Windows 7 lan kosok balene kasedhiya ing Windows Live Downloads.[1]
Windows Live Essentials disepadukan dengan Windows 7 tetapi tidak dibungkuus secara bersama dalam Windows 7 dan sebaliknya tersedia di Windows Live Downloads.[1]
