Windows 7 został oznaczony nazwą kodową „Windows NT 6.1”.
Windows 7-ga "Windows NT 6.1" nomi berilgan.
Oznaczenie to ma na celu zwiększenie kompatybilności programów napisanych na poprzednie wersje systemu z Windows 7[5][6].
Ushbu belgi tizimning oldingi versiyalari uchun yozilgan dasturlarning Windows 7 bilan mosligini oshirish uchun mo'ljallangan[1][2].
Microsoft argumentował decyzję tym, że wiele programów sprawdza czy numer wersji mieści się w pewnym przedziale.
Microsoft ko'plab dasturlar versiya raqami ma'lum bir diapazonda ekanligini tekshiradi deb ta'kidladi.
Mogą się one nie uruchomić jeśli liczba ta będzie za duża, mimo że sam program działałby poprawnie w nowym środowisku[6].
Agar ularning soni juda katta bo'lsa, ular boshlamasligi mumkin, garchi dasturning o'zi yangi muhitda to'g'ri ishlaydi[2].
Firma zaprzeczyła też, że niewielka zmiana w oznaczeniu wersji oznacza brak istotnych różnic między systemem Windows 7 a Windows Vista (oznaczonego numerem „6.0”)[5].
Kompaniya, shuningdek, versiyalarni belgilashdagi ozgina o'zgarishlar Windows 7 va Windows Vista ("6.0" raqami bilan belgilanadi) o'rtasida jiddiy tafovutlar yo'qligini anglatadi[1].
Podobna sytuacja miała miejsce w przypadku Windows XP.
Shunga o'xshash vaziyat Windows XP-da ham yuz berdi.
Windows 2000 oznaczony był numerem 5.0, a XP, mimo poważnych zmian, wydany został „tylko” jako wersja 5.1[1].
Windows 2000 raqami 5.0 edi ko'rsatilgan va XP, muhim o'zgarishlarga qaramay, bir versiya 5.1 sifatida "faqat" ozod qilindi[1].

Jako zawodnik grał w Spartaku Samarkanda, ale szybko porzucił karierę piłkarską na rzecz pracy szkoleniowej.
