مردم سیستانی (از لحاظ تاریخی به «سکزی» نیز گفته می‌شود).[1] یکی از اقوام ایرانی‌تبار هستند[2][3][4] که عمدتاً در منطقه‌ای به نام سیستان در جنوب‌شرقی ایران و از لحاظ تاریخی جنوب‌غربی افغانستان ساکن هستند.[5]زبانشان فارسی و گویش سیستانی می‌باشد.[6]
Imezdaɣ n Sista (tessemman-itt daɣen s ⁇ skzi ⁇).[1] D yiwen seg yimezdaɣ n Tbar[2][3][4] i deg qqimen deg tmazirt n Sistan deg ugafa n tmurt n Iran, d ugur-is n ugafa n Afɣani, d agdud n tmurt n Tbar.[5]Tmeslayt-nsen d tafransist, d tasusmiwt n Sistine.[6]

سیستانی‌ها نام خود را از سکاستان ("سرزمین ساکا ") گرفته‌اند. ساکاها از قبیله سکاها بودند که به فلات ایران مهاجرت کردند.[ نیاز به نقل از ] نام قدیمی‌تر پارسی باستان منطقه - قبل از تسلط ساکاها - Zaranka یا Drangiana ("سرزمین آبی") بود.[ نیاز به منبع ] این شکل قدیمی‌تر ریشه نام زرنج مرکز استان نیمروز افغانستان نیز می‌باشد.

مردم سیستانی (از لحاظ تاریخی به «سکزی» نیز گفته می‌شود)، یکی از اقوام ایرانی‌تبار هستند[1][2][3][4] که عمدتاً در منطقه‌ای به نام سیستان در جنوب‌شرقی ایران و از لحاظ تاریخی جنوب‌غربی افغانستان ساکن هستند.[5]زبانشان فارسی و گویش سیستانی می‌باشد.[6]
Agdud n Sistan (s umezruy yettusemman daɣen s ⁇ skzi ⁇), d yiwen seg yimdanen n Tbar Iranit[1][2][3][4] i deg deg iqqim deg temnaḍt n Sistan deg ugafa n tmurt n Iran, d ugdud n ugafa n ugafa-d n tmurt n Afɣani.[5]Tmeslayt-nsen d tafransist, d tasusmiwt n Sistine.[6]
راولینسون به لحاظ نژادی، سیستانی‌ها را همراه با جمشیدی‌های هرات، نمونه خالص نژاد آریایی می‌داند؛[1][2]
Rawlinson yeqqar-d s wudem n tawsit, i cistayen, akked iẓuran n Ḥerrat, d amedya n tuɣalin n tirga n Arya؛[1][2]
مردم سیستان در گذشته به گویش‌های میانه ایرانی چون پهلوی اشکانی، پارسی میانه (پهلوی ساسانی) صحبت می‌کردند[1] و اکنون به گویشی از زبان فارسی که به سیستانی معروف است، صحبت می‌کنند. سیستانی‌ها از بازماندگان اقوام سکایی هستند.[2] سکاها آخرین دسته از آریایی‌هایی بودند که در سال ۱۲۸ ق.م وارد ایران شدند.[3][4][5] آنها در بخش‌های مرکزی و شمالی استان سیستان و بلوچستان زندگی می‌کنند. از دهه‌های اخیر بسیاری نیز به دلایل مختلف سیاسی و اقلیمی به سایر نقاط ایران مانند استان‌های تهران و گلستان در شمال ایران مهاجرت کرده‌اند.[6]
Imdanen n Sistan qqaren-d zik s tutlayin n temẓiwin n Tmazight n Tmazɣa n Tmazayt am Peluwi Ashkani, Peluwi Tasani d tin n Tmazyazt n Tmazlayt n Tmazyag n Tmazɣt i d-yettwassnen s Tmazight. Iṣṣṣan d imeddukal n tmetti n Suki.[2]Isekkasen d tiɣremt taneggarut n Yiṛayaniyen i d-yettwakksen deg useggas n 128 n jmiɛ n Lmasiḥ.Imdanen n Meḥṭan rran-d deg Iran.[3][4][5] Zgan deg temnaḍt n temnaḍin n Sistan d Baluchistan. Seg iseggasen-a ineggura, aṭas n medden i d-yeddun ɣer tmura nniḍen n Iran, am tegzirt n Tehran d Gulistan deg ugafa n tmurt, ɣef ddemma n yal yiwet n tɣawsa tasertant d temnaḍt.[6]
