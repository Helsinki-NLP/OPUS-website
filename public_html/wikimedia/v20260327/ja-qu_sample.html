Windows Copilot（ウィンドウズ・コパイロット）は、Microsoft社がWindows 11に組み込むために開発した生成AIである。
Windows Copilot nisqaqa huk paqarichisqa KY Microsoft kaqwan ruwasqa Windows 11 kaqpi churanapaq.

概要
Tukuy imamanta qhawariy
2023年5月23日（米国時間）、Microsoft社は、アメリカ合衆国本社とYouTube上で、年次イベント「Microsoft Build」を公開し、Windows 11に生成AIを組み込んだWindows Copilot（公式名称：Windows Copilot for Windows 11）を発表した[1]。Windows Copilotは、PCのデスクトップ画面でAIと対話し、OSの設定変更やアプリケーションの操作、文章の要約などをAIがサポートする。2023年6月からプレビュー版を提供する予定である。
23 ñiqin Qhulla puquy killapi 2023 watapi (HAM pachapi), Microsoft sapa wata ruwayninta "Microsoft Build" HAM umalliq wasipi chaymanta YouTube kaqpi lluqsichirqan, chaymanta Windows Copilot (tukri suti: Windows Copilot willasqa Windows 11 kaqpaq) lluqsichirqan.[1] Windows Copilot AI kaqwan PC hamp'ara qhawanapampapi ruwaypurakuq, chaymanta KY llamk'aykuna llikachap mast'inankunata t'ikrayta atin, ruwachina wakichikuna, rimaykuna ima qillqa pisiyachiyninpi ruwayta yanapan. Huk ñawpaq rikuy niraqta Chakraqunakuy killamanta 2023 quyta musikachkanku.
