 <s id="1">2.9.1.2 Generalized Switching Fabric</s>
 <s id="2">In the case of generalized switching fabric one can make all types of input/ output connections, i.e. point‐to‐point, point‐multipoint (1 × N), or multicasting (N × 1).</s>
 <s id="3">In general, it is an (N × R) matrix with 2NR connecting states.</s>
 <s id="4">Though the permutation switch has only one point closing in a row or column of the connection matrix, for generalized fabric switch there can be multiple points closing in a row or column depending on whether it has broadcasting or multicasting connections.</s>
 <s id="5">Unlike the permutation switch, which has only the control for bar and cross state, the generalized matrix has additional controls to specify the number of connections, as shown in Figure 2.20b for a 4 × 4 matrix.</s>
 <s id="6">2.9.1.3 Linear Divider and Combiner Switching Fabric</s>
 <s id="7">2.9.1.1 Permutation Switching Fabric</s>
 <s id="8">The interconnection pattern between the different input ports to the output ports of the switching fabric is represented by a connection matrix.</s>
 <s id="9">The interconnection between the switches in the case of N × N permutation switching fabric is such that there can be only point‐to‐point (one input port to another output port) unicast connection.</s>
 <s id="10">It cannot have broadcasting (1 × N) or multicasting (N × 1) connections.</s>
 <s id="11">The permutation fabric matrix, therefore, will have one cross‐point closing in each row and column.</s>
 <s id="12">A connection matrix for the 4 × 4 permutation fabric is shown in Figure 2.20a.</s>
 <s id="13">The switching matrix is square in this case as there can be only permutation connections possible.</s>
 <s id="14">The total possible input/output connection patterns or states are N! in a permutation switch fabric.</s>
 <s id="15">In the generalized switching fabric when we have the combining and splitting ratios also defined for multicasting and broadcasting, respectively, they ar known as linear divider and combiner (LDC) switching.</s>
 <s id="16">As the combiner/splitting ratios can take any value between 0 and 1, therefore, LDC can have a continuum of connection states.</s>

 <s id="18">2.9.2 Classification According to Blocking Characteristics</s>
 <s id="19">In any network a very large number of switches cannot be made available to provide dedicated connections to all possible connections required.</s>
 <s id="20">This is done in order to provide better utilization of the resources but without compromising on the performance of the services.</s>
 <s id="21">Therefore, with the prior estimate of the traffic on the network, they are designed that they have no or least little blocking to the admission of requests with the available number of switches.</s>
 <s id="22">So the switches we consider here are non‐blocking switches and are classified according to their blocking characteristics for randomly made requests to the network for the connections between any input and any output port.</s>
 <s id="23">According to the blocking of the randomly appearing input requests, the switch fabrics can be of the following types.</s>
