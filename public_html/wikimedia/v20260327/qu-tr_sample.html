Open GXpa riqsinapaq yupin Opera GX nisqaqa huk pukllayman-qhawarisqa niraq Opera web wamp'uq Microsoft Windows kaqpaq chaymanta MacOS kaqpaq, Opera Llamp'ukaq kaqwan ruwasqa.
Opera GX, Opera Software tarafından geliştirilen, Microsoft Windows ve MacOS için Opera web tarayıcısının oyun odaklı bir versiyonudur.

Tiksimuyuntinpi ñawpaq pukllay wamp'uqmi.
Dünyanın ilk oyun tarayıcısıdır.
Antañiqiq pukllay munakuqkunapaq ruwasqa.
Bilgisayar oyunu severlerine yönelik geliştirilmiştir.
Opera GX nisqaqa 11 ñiqin qhapaq raymi killapi 2019 watapi willasqa karqan, E3 2019 qallarisqan p'unchawllapi.
Opera GX, E3 2019'un başlangıcı ile aynı gün olan 11 Haziran 2019'da duyuruldu.
