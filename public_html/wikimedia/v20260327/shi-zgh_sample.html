Farid Muḥmmad Zalḥuḍ, ilul ɣ usggʷas n 1959 (asakud n tlalit n tidt 1957) ɣ usun n Waday, tiɣrt n Tfrawt, tasga n Tznit, ḍaṛt ma ismd tiɣri tamnzut ɣ tinml n Muḥmmad Wis Smmus da nit illan ɣ tmazirt nns, iḍfṛ tiɣri nns ɣ usinan n mulay ṛcid ɣ tznit, yamẓ gis aslkin n lbakaluṛya ɣ tskla asggʷas n 1977. Ḥamid inrzaf (nɣ d s ism nns aẓɣuṛan ḥamid bayḥ, s tɛṛabt: حميد إنرزاف، s tfṛanṣIṣt: hamid inerzaf, s uskkil alatin n tmaziɣt: ḥamid inrzaf) iga amɣnnj amaziɣ amɣribi, ig anḍḍam amaziɣ amqqṛan. Ilul ḥamid inrzaf ɣ usggʷas n 1972 ɣ usun n sswalm, actukn, maca yucka d s bnsrgaw zɣ lliɣ dars 2 isggʷasn ɣ uwtay, ilmd ḥamid inrzaf aẓawan d tẓuṛI ɣ tgmmi n ḥasan aɛṭuṛ zun kigan n inuẓaṛ. Ikka ḥamid inrzaf tugt n trubba am tarabbut n "Lmsabiḥ", tarabbut n "Laḥnan", tarabbut n "Iqqbiln" ɣ usggʷas n 1984 d trabbut yusin nit ism ns "Inrzaf" .
