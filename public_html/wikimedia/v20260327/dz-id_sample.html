Microsoft Windows ཡང་ན་ Windows ཟེར་མི་འདི་ ཡོངས་གྲགས་ཅན་ཅིག་ཨིན་མི་ operating system བཟའ་ཚང་ཅིག་ཨིན་མི་ Microsoft གིས་བཟོ་སྐྲུན་འབད་ཡོདཔ་ཨིན། འདི་ཡང་ GUI ལག་ལེན་འཐབ་སྟེ་ཨིན།
Microsoft Windows atau yang lebih dikenal dengan sebutan Windows adalah keluarga sistem operasi yang dikembangkan oleh Microsoft, dengan menggunakan antarmuka pengguna grafis (GUI).

Windows ལས་རིམ་འདི་ MS-DOS ལས་ཐོན་ཡོདཔ་ཨིན། འདི་ཡང་ ཡིག་ཆ་དང་བཀའ་བཀོད་གི་ཐིག་ཁྲམ་ལུ་གཞི་བཞག་སྟེ་ཨིན། Windows གི་ཨང་དང་པ་ཨིན་མི་ Windows Graphic Environment ༡.༠ པ་དེ་ སྤྱི་ལོ་༡༩༨༣ ཟླ་༡༡ པའི་ཚེས་༡༠ ལུ་ འགོ་དང་པ་བཟོ་ཡོདཔ་ཨིན་རུང་ སྤྱི་ལོ་༡༩༩༨༥ ཟླ་༡༡ ལུ་ ཚོང་འབྲེལ་འཐབ་པའི་བསྒང་ལས་ བཟོ་སྐྲུན་འབད་མི་དེ་ བཟོ་སྐྲན་འབད་ནིའི་དོན་ལས་ བཟོ་རིག་གི་གནས་སྟངས་ལུ་བརྟེན་ཨིན། Windows 1.0 འདི་ MS-DOS (དང་ MS-DOS གི་འགྱུར་བཅོས་དག་པ་ཅིག་) གྱི་ཐོག་ལུ་ལཱ་འབད་ཡོད་པའི་ ཨེཕ་ཊ་ ༡༦-bit འབད་མི་ སྣུམ་འཁོར་ཆས་ཅིག་ཨིནམ་ལས་ འདི་གི་དོན་ལུ་ DOS ལས་འཛིན་མེད་པ་ཅིན་ འདི་ཡང་མ་འགྲོ་བར་འགྱོ་ཚུགས། འདྲ་བཤུས་ 2.x དང་ 3.x འདི་ཡང་འདྲ་མཉམ་ཨིན། མཐའ་མའི་ Windows གྱི་འདྲ་ཕབ་དག་པ་ཅིག་ (Windows 4.0 དང་ Windows NT 3.1 ལས་འགོ་བཙུགས་ཏེ་) འདི་རང་དབང་ཅན་གྱི་ལག་ལེན་ལམ་ལུགས་ཅིག་ཨིནམ་ལས་ ད་ལྟོའི་བར་ན་ཡང་ MS-DOS གྱི་ལག་ལེན་ལམ་རིམ་ལུ་བརྟེན་མི་ཚུགས། དེ་ལས་ Microsoft Windows འདི་ཡར་རྒྱས་འགྱོ་སྟེ་ ཌེསི་ཊེམ་གྱི་ལག་ལེན་དེ་ བརྒྱ་ཆ་༩༠ ལུ་ ཡར་སེང་འགྱོ་སྟེ་ཡོདཔ་ཨིན།
Sistem operasi Windows telah berevolusi dari MS-DOS, sebuah sistem operasi yang berbasis modus teks dan command-line. Windows versi pertama, Windows Graphic Environment 1.0 pertama kali diperkenalkan pada 10 November 1983, tetapi baru keluar di pasaran pada bulan November tahun 1985, yang dibuat untuk memenuhi kebutuhan komputer dengan tampilan bergambar. Windows 1.0 merupakan perangkat lunak 16-bit tambahan yang berjalan di atas MS-DOS (dan beberapa varian dari MS-DOS), sehingga ia tidak akan dapat berjalan tanpa adanya sistem operasi DOS.
Microsoft Corporationའདི་ Redmond, Washington, United Statesལུ་གཞི་བཙུགས་འབད་མི་ ཡུ་ཨེསི་ཌ་གི་རྒྱལ་སྤྱི་ཚོགས་སྡེ་ཅིག་ཨིནམ་ད་ དེ་གིས་ གློག་རིག་དང་འབྲེལ་བའི་ ཅ་ཆས་དང་ཞབས་ཏོག་ཚུ་ གོང་འཕེལ་གཏང་ནི་དང་ བཟོ་སྐྲུན་འབད་ནི་དང་ ལི་སི་ལེན་འབད་ནི་ དེ་ལས་ རྒྱབ་སྐྱོར་འབད་ཡོདཔ་ཨིན། ལས་སྡེ་འདི་ Bill Gates དང་ Paul Allen གིས་ སྤྱི་ལོ་༡༩༧༥ སྤྱི་ཟླ་༤ པའི་ཚེས་༤ ལུ་ གཞི་བཙུགས་འབད་ནུག Microsoft འདི་འཛམ་གླིང་གི་ཐོན་སྐྱེད་ལས་སྡེ་སྦོམ་ཤོས་ཅིག་ཨིན་མས།[1] Microsoft འདི་ཡང་ འཛམ་གླིང་ནང་གི་ ཚོང་འབྲེལ་འཐབ་མི་ ལས་སྡེ་ཚུ་གི་གྲལ་ལས་ཨིན།[2]
Microsoft Corporation adalah perusahaan multinasional Amerika Serikat yang berpusat di Redmond, Washington, Amerika Serikat yang mengembangkan, membuat, memberi lisensi, dan mendukung berbagai produk dan jasa terkait dengan komputer. Perusahaan ini didirikan oleh Bill Gates dan Paul Allen pada tanggal 4 April 1975. Microsoft merupakan pembuat perangkat lunak terbesar di dunia menurut pendapatannya.[1] Microsoft juga merupakan salah satu perusahaan paling bernilai di dunia.[2]
Microsoft གིས་ གཞི་བཙུགས་འབད་མི་དེ་ Altair 8800 གི་དོན་ལུ་ BASIC སྐད་བསྒྱུར་པ་བཟོ་ནི་དང་བཙོང་ནི་ཨིན་མས། ལས་སྡེ་འདི་གིས་ སྤྱི་ལོ་༡༩༨༠ གི་བར་ན་ རང་སོའི་ཀོ་བིཌ་-༡༩༨༨ གི་ MS-DOS ཌའི་ལོག་ལས་རིམ་གྱི་ཚོང་འབྲེལ་ལུ་དབང་བཟུང་འབད་ཡོདཔ་ད་ འདི་གི་ཤུལ་ལས་ Microsoft Windows ཌའི་པར་གྱིས་ ཌའི་ཀར་ལས་རིམ་ཚུ་བཟོ་སྟེ་ཡོདཔ་ཨིན། སྤྱི་ལོ་༡༩༨༦ ལུ་ Microsoftགི་འགོ་འདྲེན་འཐབ་མི་ སྤྱིར་བཏང་བཙོང་སྣེ་དང་ ཚོང་འབྲེལ་གྱི་གོང་ཚད་ཡར་སེང་གིས་ Microsoftགི་ལས་བྱེདཔ་ཚུ་གི་གྲལ་ལས་ མི་ཡ་༣ དང་ མི་ཡ་༡༢,༠༠༠ བཟོ་སྐྲུན་འབད་ཡོདཔ་ཨིན། སྤྱི་ལོ་༡༩༩༠ ལོའི་སྐབས་ལས་ ལས་འཛིན་དེ་ ལས་འཛིན་གྱི་ཚོང་ལས་སྣ་མང་སུ་སྦེ་ར་ གྱུར་ཏེ་ཡོདཔ་ལས་ ལས་འཛིན་གྱི་ཉོ་ལེན་ལེ་ཤ་འབད་ཡོདཔ་ཨིན། སྤྱི་ལོ་༡༩༡༡ ལུ་ Microsoft གིས་ Skype Technologies ཟེར་མི་ལས་སྡེ་འདི་ དངུལ་ཀྲམ་ཐེར་འབུམ་༨.༥ ལུ་ཉོ་སྟེ་ཡོདཔ་ད་ འདི་གིས་ Microsoft གི་ལོ་རྒྱུས་ནང་ གོང་ཚད་མཐོ་ཤོས་ཅིག་སྦེ་ བཟོ་ཚུགས་ནུག[1]
Perusahaan ini berhasil mendominasi pasar sistem operasi komputer pribadi dengan MS-DOS nya pada pertengahan 1980-an, diikuti dengan jajaran sistem operasi Microsoft Windows. Penawaran umum perdana Microsoft tahun 1986, dan kenaikan tajam harga sahamnya, menciptakan tiga miliuner dan 12.000 jutawan di kalangan karyawan Microsoft. Sejak 1990-an, perusahaan ini semakin terdiversifikasi dari pasar sistem operasi dan telah melakukan sejumlah akuisisi perusahaan.
སྤྱི་ལོ་༢༠༠༢ ལུ་ Microsoft གིས་ PC གི་ལག་ལེན་གྱི་ལམ་ལུགས་དང་ ཡིག་ཚང་གི་མཉེན་ཆས་ཀྱི་ཚོང་འབྲེལ་ (Microsoft Office གི་མཉམ་འབྲེལ་) ཚུ་ནང་དབང་འཛིན་འབད་ཡོདཔ་ཨིན། ལས་སྡེ་འདི་གིས་ ད་རུང་ཌེཀསི་ཊོཔ་དང་ ཞོག་རྒྱུན་གྱི་དོན་ལུ་ ཞབས་ཏོག་གི་རིགས་ཅིག་ བཟོ་སྐྲུན་འབད་ཡོདཔ་ད་ འདི་ཡང་ ཡོངས་འབྲེལ་འཚོལ་ (Bing) གྱི་ས་ཁོངས་ཚུ་ནང་ ལཱ་འབད་ཡོདཔ་ད་ གློག་བརྙན་རྩེད་རིགས་ (Xbox དང་ Xbox 360 console) ཚུ་དང་ ཌི་ཇི་ཊི་ཨེསི་ཊི་ཨེན (MSN) ཚུ་ དེ་ལས་ འགྲུལ་འཕྲིན་ལག་ལེན་ (Windows Phone OS) ཚུ་ནང་ ལཱ་འབདཝ་ཨིན་པས། སྤྱི་ལོ་༢༠༠༢ ཟླ་༦ ལུ་ Microsoft གིས་ PC ཝེན་ཌོ་ར་ཚོང་འབྲེལ་འཐབ་མི་ཚུ་ལུ་ འགོ་དང་པ་སྦེ་ Microsoft Surface གྱི་ས་ཆ་རྩིས་ཆས་ཚུ་ འགོ་བཙུགས་ཐོག་ལས་ ཌོ་ཊ་ཨེམ་ཚོང་འབྲེལ་འཐབ་ནི་ཨིནམ་སྦེ་ ཁྱབ་བསྒྲགས་འབད་ནུག།
Tahun 2012, Microsoft adalah pendominasi pasar sistem operasi PC dan pasar perangkat lunak perkantoran (bersama Microsoft Office). Perusahaan ini juga memproduksi serangkaian perangkat lunak untuk desktop dan server, dan aktif di sejumlah bidang seperti pencarian Internet (Bing), industri permainan video (konsol Xbox dan Xbox 360), pasar layanan digital (MSN), dan telepon genggam (Windows Phone OS). Bulan Juni 2012, Microsoft mengumumkan bahwa mereka akan memasuki pasar vendor PC untuk pertama kalinya melalui peluncuran komputer tablet Microsoft Surface.
སྤྱི་ལོ་༡༩༩༠ ལོར་ལུ་ དབྱེ་ཞིབ་པ་ཚུ་གིས་ མདོ་དམ་འཐབ་མི་ལས་སྡེ་ Microsoft གིས་ མནའ་རྟགས་ཅན་གྱི་ཚོང་འབྲེལ་གྱི་ལམ་ལུགས་དང་ དོ་འགྲན་བཀག་སྡོམ་ཀྱི་ཐབས་ལམ་ཚུ་ ལག་ལེན་འཐབ་སྟེ་ཡོདཔ་ལས་ ངོས་ལེན་དང་བཅའ་ཡིག་ཚུ་ མནའ་ལོག་མ་གཏང་ནི་དང་ ཞབས་ཏོག་ལག་ལེན་ལུ་ ཚུལ་མིན་གྱི་ཚད་འཛིན་ཚུ་ འབད་ནི་ དེ་ལས་ ངོ་ཚབ་མ་སྦེ་ ཚོང་འབྲེལ་འཐབ་ནི་ཚུ་ འབད་དོ་ཡོདཔ་ལས་ US ཁྲིམས་སྲུང་འགག་པ་དང་ ཡུ་རོབ་ཚོགས་པའི་ལས་སྡེ་ཚུ་གིས་ ལས་སྡེ་འདི་ ངན་ལྷད་བཀག་སྡོམ་འབད་ཡོད་པའི་བཅའ་ཁྲིམས་དང་འགལ་བའི་བཅའ་ཁྲིམས་ཚུ་ བཀལ་ནུག། ལས་སྡེ་འདི་ Apple Inc., Amazon.com, Alphabet, དང་ Facebookདང་གཅིག་ཁར་ Big Four གི་ནང་འཁོད་ལུ་རྩིས་ཏེ་ཡོདཔ་ཨིན།[1][2]
Pada tahun 1990-an, para kritikus mulai menuduh bahwa Microsoft menjalankan praktek bisnis monopolistik dan strategi anti-persaingan, termasuk penolakan persetujuan dan pengikatan, membuat batasan yang tidak masuk akal dalam penggunaan perangkat lunaknya, dan melakukan taktik pemasaran yang tidak representatif; baik Departemen Kehakiman AS dan Komisi Eropa menyatakan perusahaan ini melanggar hukum antitrust. Perusahaan ini dianggap sebagai salah satu perusahaan teknologi Big Four bersama dengan Apple Inc., Amazon.com, Alphabet, dan Facebook.[1][2]
Tibetan ཡི་གེ་འདི་ Brahmi ཡི་གེ་ལས་ འབྱུང་ཁུངས་ abugidaཅིག་ཨིན་ འདི་ Tibetan, Dzongkha, Sikkim, Ladakhi, དང་དུས་སྐབོད་སྐད་ Balti ཡི་གེ་ཚུ་འབྲི་ནིའི་དོན་ལུ་ལག་ལེན་འཐབ་ཡོདཔ་ཨིན། ཌི་ཨེམ་ འབྲུག་གི་ཨེབ་གཏང་འབད་ཡོད་མི་ ཨེབ་གཏང་འབདཝ་ཨིན་ (Tibetan: དབུ་ཅན་; Wylie: dbu-can; "མགུ་ཏོ་ཡོད་མི་") དེ་ལས་ལག་ཡིག་གི་ལག་ཡིག་གི་ཨེབ་ཐག་འདི་ ཨེབ་གཏང་འབད་མི་ ཨེབ་གཏང་འབདཝ་ཨིན། འདི་ཡང་ ཊི་བེ་ཊི་གི་ མི་བརྒྱུད་ཀྱི་ ངོ་ཤེས་དང་འབྲེལ་བ་ཡོདཔ་ཨིན། [1] ཐིམ་ཕུག་ལུ་ཡང་ ཐིམ་ཕུག་གི་སྐད་ཡིག་ཚུ་ནང་ ཐིམ་ཕུའི་སྐད་ཡིག་ཚུ་ ལག་ལེན་འཐབ་དོ་ཡོདཔ་ད་ རྒྱ་གར་ ནེ་པཱལ་ ཐི་བེ་ཊི་གི་སྐད་ཡིག་གི་མིང་ཡང་ ལག་ལེན་འཐབ་དོ་ཡོདཔ་ཨིན། Tibetan [2] ཡིག་འབྲུ་གཞན་ཚུ་བཀོད་ནི་ལུ་ ཕན་ཐོགས་བྱུང་ནུག དཔེར་ན་ Lepcha ཡིག་འབྲུ་དང་ 'Phags-pa ཡིག་འབྲུ་ཚུ་བཀོད་ཡོདཔ་ཨིན།
Bentuk cetak dari aksara ini disebut aksara uchen (Tibet: དབུ་ཅན་; Wylie: dbu-can; "dengan kepala") sedangkan bentuk tulisan tangan yang digunakan dalam tulisan sehari-hari disebut aksara umé (Tibet: དབུ་མེད་; Wylie: dbu-med; "tanpa kepala"). Aksara ini berhubungan erat dengan identitas etnis Tibet yang luas. Selain di Tibet, sistem penulisan ini juga digunakan untuk bahasa-bahasa Tibet di Bhutan dan sebagian India dan Nepal bahkan Pakistan.[1] Aksara Tibet memengaruhi terciptanya aksara-aksara lain, seperti diantaranya aksara Lepcha dan aksara 'Phags-pa.[2]
