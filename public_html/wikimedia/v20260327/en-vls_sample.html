Riot.im is an open source, free software instant messaging client based on the Matrix protocol and is distributed under the free software Apache License 2.0.
Riot.im is e chatprogramma gebaseerd ip ’t Matrix-protocol en wordt uutgegeevn ounder d’Apache-licensje 2.0, e vrije software-licensje.
Because it uses the federated Matrix protocol, Riot.im lets the user choose a server to connect to.[5]
Met da Riot ’t gefedereerd Matrix-protocol gebruukt, kut de gebruker kiezn welke server da t’n wilt gebruukn.
Additionally, Riot.im supports end to end encryption, groups, channels and sharing of files between users,[6] also registration does not require a phone number.[7] It is available as a web application, as desktop apps for all major operating systems and as a mobile app for Android and iOS.[8] The development of the app is primarily done by the company New Vector Limited[9], which is also involved in the development of the Matrix protocol itself.[10]
Riot oundersteunt ook end-tout-end-versloterienge, groepn, kanoaln en bestandn deeln met andere gebrukers.[1] J’en è geen telefongnummer nodig vo je te registreern.[2] Riot is beschikboar as e webtoepassienge, as a geweun programma vo Windows, [[MacOS]] en Linux en as een app vo Android en iOS.[3] D’ountwikkelienge is vo ’t grotste deel in handn van ’t bedryf New Vector Limited,[4] dat ook vele werkt aan d’ountwikkelienge van ’t Matrix-protocol zelve.[5]
Technology
Technologie
Riot.im is built with the Matrix React SDK[11], which is a React based software development kit to ease the development of Matrix clients.
Riot.im is gemakt met de Matrix React SDK[1], een ip React gebaseerde softwareountwikkeliengskit vo d’ountwikkelienge van Matrix-cliëntn gemakkeliker te moakn.
Riot.im is mostly built around web technologies and also uses Electron, a software framework to create desktop applications from web applications, to distribute their desktop clients for Windows, MacOS and Linux.
Riot.im is voor e grot stik gebouwd ip webtechnologieën en makt ook gebruuk van Electron, e softwareroamwerk vo bureaubladtoepassiengn te moakn an d’hand va webtoepassiengn, vo de programma’s vo Windows, MacOS en Linux.
The Android and iOS clients are developed and distributed with their respective platform tools.
D’apps vo Android en iOS wordn ontwikkeld me ’t specifiek gereedschap voun hunder platforms.
On Android the app is available both in the Google Play Store[12] and the free-software only F-Droid[13] Archives, with minor modifications. For instance the F-Droid version doesn't come with the proprietary Google Cloud Messaging plug-in.
Ip Android is den app beschikboar in de Google Play Store[1] en F-Droid.[2] De versie van F-Droid èt e poar anpassiengn en èt bijvoorbeeld geen propriëtaire plug-in vo Google Cloud Messaging.
