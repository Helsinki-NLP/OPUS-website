Milestone MS3 - Resource repository

9 years late (Aug 30, 2011, 10:00:00 PM)

Resource repository populated with initial training data

  • leading partner: UUP
  • WP2

Training data for 10 language pairs and 3 domains are collected and processed, D2.3

Sub-goal 1: Robust REST API

  • completely implemented REST API for LetsMT
  • documentation of REST requests
  • documentation of setup & administration
  • running REST server for experiments

Sub-goal 2: Demo-interface to repository prototype

  • access to LetsMT repository
  • simple web interface for several tasks
    • login
    • create resource
    • upload resources (TMX)
    • browsing resources and searching for specific languages & language pairs / domains, ...
  • selecting resources for SMT training
