Installation and Setup

  • Install the complete LetsMT repository software ( sudo make install) or the LetsMT RR frontend (sudo make install-frontend)
  • Install the interface 
sudo make install-grenzschnitte
  • adjust the mysql user and password in /usr/local/lib/grenzschnitte/web_interface.json
  • check that the interface is activated in Apache (/etc/apache2/sites-enabled/grenzschnitte) and adjust the proxy port if you like (standard = 81)
  • adjust the firewall settings if necessary

Firewall on OPUS

OPUS uses Shorewall and the settings are in /etc/shorewall/. Rules can be set in /etc/shorewall/rules. After all changes restart with 

service shorewall restart

Troubleshooting

  • user mojo needs to be created in LetsMT-backend (system user) (should be created by grenzschnitte, but does not seem to work)
