Amazon EC2 Setup

This is a short description of how to setup, start and stop a new Ubuntu instance on the Amazon EC2 service. TOC(depth=1)?

Set up and start a new system instance

  • Point you browser to ​http://aws.amazon.com
  • Click on Sign in to the AWS Managemnt Console
  • Login in
  • Click on the EC2 tab
  • Choose Region: US East (this is currently the cheapest option)
  • Choose Instances
  • Click on Lauch Instance

In the Request Insatances Wizard:

  • Choose an AMI (Amazon Machine Image)
    • Click on the tab Community AMIs
    • Choose Viewing: Public Images and search for: ubuntu-lucid-10.04-amd64-server-2011
    • Select the entry with Root Device: instance-store
  • Instance Details
    • Number of Instances: 1
    • Availablility Zone: no preference
    • Instance Type: Large 7.5 GB
    • Termination Protection: not ticked
    • Select Launch Instance
    • Click Continue
    • Kernel ID: default
    • RAM Disk ID: default
    • Monitoring: not ticked
    • User Data: empty
    • Click Continue
    • For the key Name enter a value that identifies this instance as yours, like 'Matthias test'
    • Click Continue
  • Create Key Pair
    • choose Your existing Key Pairs: letsmt-aws-key
    • Click Continue
  • Security Group
    • Security Groups: ssh
  • Review
    • check all settings and click on Launch
    • Close the information window

Back in the My Instances window wait for the instance to change Status form pending to running.

Select the just started instance to see details in the window below. Copy the value in the Public DNS field.

Done!

To login to the new instance from opus use the following command where you replace the server name with the public DNS you just copied from your new instance: 

ssh -i /home/letsmt/letsmt-aws-key.pem ubuntu@ec2-204-236-170-135.us-west-1.compute.amazonaws.com

Stop an instance

In the Instance section of the AWS Management Console select your instance and choose Terminate from the Instance Actions menu.
