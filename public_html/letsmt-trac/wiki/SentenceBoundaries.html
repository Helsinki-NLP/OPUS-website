Tools for sentence boundary detection

  • New trainable tokenizer:
    • ​https://github.com/jirkamarsik/trainable-tokenizer
  • Europarl sentence splitter as Perl modules:
    • ​http://code.google.com/p/corpus-tools/downloads/list
    • ​http://search.cpan.org/~achimru/Lingua-Sentence-1.02/lib/Lingua/Sentence.pm
  • Other Perl modules:
    • ​http://search.cpan.org/~shlomoy/Lingua-EN-Sentence-0.25/lib/Lingua/EN/Sentence.pm
    • ​http://search.cpan.org/~holsten/Lingua-DE-Sentence-0.07/Sentence.pm
    • ​http://search.cpan.org/~shlomoy/Lingua-HE-Sentence-0.13/lib/Lingua/HE/Sentence.pm
  • Punkt
    • implemented in NLTK (Apache license)
    • ​http://www.nltk.org/
    • trainable (unsupervised)
    • ​http://www.linguistics.ruhr-uni-bochum.de/~strunk/ks2005FINAL.pdf
    • ​http://docs.huihoo.com/nltk/0.9.5/guides/tokenize.html
    • ​http://nltk.googlecode.com/svn/trunk/doc/api/nltk.tokenize.punkt-module.html
  • OpenNLP (Apache license)
    • ​http://opennlp.apache.org/
    • trainable sentence boundary detector
    • models available for English, German, Spanish, Thai
    • wiki: ​http://sourceforge.net/apps/mediawiki/opennlp/
  • other:
    • classifier based (English model included): ​http://code.google.com/p/splitta/ (​paper)
    • SRX rules: ​http://languagetool.svn.sourceforge.net/viewvc/languagetool/trunk/JLanguageTool/src/resource/segment.srx?revision=7751
    • C++ implementation of SRX rule-based: ​http://nlp.pwr.wroc.pl/redmine/projects/toki/wiki/
    • Stanford NLP java tools
    • English only: ​http://www.fleric.org.cn/pub/ss.rar
    • more links: ​http://mailman.uib.no/public/corpora/2010-August/011367.html
  • Jena NLP tools
    • ​http://www.julielab.de/Resources/Software/NLP+Tools.html
  • FreeLing (GPL)
    • ​http://www.lsi.upc.edu/~nlp/freeling
    • split rules
    • (mainly Catalan & Spanish?)
  • Alpino for Dutch (tokenization + sentence splitting)
    • ​http://www.let.rug.nl/vannoord/alp/Alpino/
  • MXPOST & MXTERMINATOR (research only!)
    • ​ftp://ftp.cis.upenn.edu/pub/adwait/jmx/
