Roadmap

Milestone: Old OPUS website in Uppsala

10 years late (Nov 14, 2010, 11:00:00 PM)

  • copy of the Groningen website
  • all services and data available at the server in Uppsala

Milestone: Consistent Data Format

10 years late (Nov 29, 2010, 11:00:00 PM)

  • all existing OPUS corpora in consistent data format
  • consistent file & data structure
  • documentation of this structure
  • some routines on how to import/make new data in that format

Milestone: Advanced OPUS website

10 years late (Dec 30, 2010, 11:00:00 PM)

  • all corpora searchable
  • improved download possibilities
  • better interfaces to data and search facilities
  • integration of parsed corpora (treebanks) as additional resource
  • additional resources (Åkes corpora)
  • improved pre-processing & annotation

Milestone: Parallel Treebank MUT

10 years late (Jan 30, 2011, 11:00:00 PM)

  • parallel treebanks
  • tools for annotation & parsing
  • tools for viewing / searching in parallel treebank
  • integration of MUT, OPUS, LetsMT (?)
Note: See TracRoadmap for help on using the roadmap.
