2. วิธีการดูข้อมูลฮาร์ดแวร์ลินุกซ์.
Indlela ukubona ulwazi zekhompyutha Linux.
คุณสามารถใช้ lshw-เครื่องมือในการรวบรวมข้อมูลที่ดีเกี่ยวกับส่วนประกอบฮาร์ดแวร์ของคุณ, ตัวอย่างเช่นซีพียู, ดิสก์, หน่วยความจำ, ตัวควบคุม USB ฯลฯ.
Ungasebenzisa lshw-isixhobo ukuba ukuqokelela ulwazi malunga amacandelo yentsimbi ye yakho, umzekelo CPU, lwe-, memory, nabaqondisi USB njl.
lshw เป็นเครื่องมือที่มีขนาดค่อนข้างเล็ก, และมีตัวเลือกไม่กี่คุณสามารถใช้กับมันในขณะที่การขยายตัวของข้อมูล. ข้อมูลที่ให้ไว้โดย lshw รวมตัวกันในรูปแบบที่แตกต่างกัน / proc-filer.
lshw sisixhobo encinci, kwaye kukho iindlela ezimbalwa ungasebenzisa nayo ngoxa ekwandiseni ulwazi. Ingcaciso enikwe lshw bahlanganisana ndawonye bakha ezahlukeneyo / proc-filer.
สำหรับพิมพ์ข้อมูลเกี่ยวกับฮาร์ดแวร์ลินุกซ์, เรียกใช้คำสั่งนี้.
Ukuze ufumane inkcazelo print malunga Linux hardware, Qhuba lo myalelo.
คุณสามารถพิมพ์สรุปข้อมูลฮาร์ดแวร์โดยใช้ตัวเลือกเจือ.
Uyakwazi uprinte isishwankathelo iinkcukacha zekhompyutha kukudliwa ukhetho shinyi.
ถ้าคุณต้องการสร้างการส่งออกเป็นไฟล์ HTML, คุณสามารถใช้ตัวเลือก -html.
Ukuba ufuna imveliso wokuvelisa njenge fayile HTML, ungasebenzisa i ukhetho -html.

3. วิธีการแสดงข้อมูล CPU ลินุกซ์.
Indlela ukubonisa ulwazi Linux CPU.
ดูข้อมูลเกี่ยวกับ CPU, ใช้คำสั่ง lscpu, เมื่อมันจะแสดงข้อมูลเกี่ยวกับสถาปัตยกรรมของ CPU, ยกตัวอย่างเช่นจำนวนของซีพียู, เมล็ด, CPU-familiemodell, CPU ซ่อน, ผู้ประกอบการค้า, ฯลฯ จาก sysfs และ / proc / cpuinfo.
Ukujonga ulwazi malunga CPU, ukusebenzisa umyalelo lscpu, xa ubonisa ulwazi malunga ukwakhiwa CPU, Umzekelo, inani CPUs, Iinkozo, CPU-familiemodell, CPU-nesikhumba, umthengisi, etc evela sysfs kunye / proc / cpuinfo.
อุปกรณ์ป้องกันเป็นอุปกรณ์จัดเก็บข้อมูลเช่นฮาร์ดไดรฟ์, แฟลชไดรฟ์ ฯลฯ. คำสั่ง lsblk จะใช้ในการรายงานข้อมูลเกี่ยวกับอุปกรณ์การปิดกั้นการดังต่อไปนี้.
izixhobo ibhloko izixhobo zokugcina ezifana nzima drives, drives flash etc. lsblk umyalelo isetyenziselwa ukuxela ulwazi malunga izixhobo uvimba ngolu hlobo lulandelayo.
หากต้องการดูอุปกรณ์ทั้งหมดที่ปิดกั้นในระบบ, กล่าวว่ารวมถึงตัวเลือก -a.
Ukuze ubone zonke izixhobo uvimba kwisistim, Wathi, zibandakanya ukhetho -a.
lsusb คำสั่งที่ใช้ในการรายงานข้อมูลเกี่ยวกับตัวควบคุม USB และอุปกรณ์ทั้งหมดเชื่อมต่อกับพวกเขา.
Le lsusb umyalelo osetyenzisiweyo ukuxela ulwazi malunga izilawuli USB kwaye zonke izixhobo enxulumene kubo.
คุณสามารถใช้ตัวเลือก v เพื่อสร้างข้อมูลรายละเอียดเกี่ยวกับแต่ละอุปกรณ์ USB.
Uyakwazi ukusebenzisa ukhetho -v ukujikelezisa inkcazelo eneenkcukacha malunga isixhobo ngasinye USB.
อุปกรณ์ PCI อาจรวมถึงพอร์ต USB, กราฟิกการ์ด, การ์ดเครือข่ายอื่น ๆ. lspci เครื่องมือที่ใช้ในการสร้างข้อมูลเกี่ยวกับตัวควบคุม PCI ในระบบ, และอุปกรณ์เชื่อมต่อกับพวกเขา.
izixhobo PCI ingafaka USB kweyakho, Card yemizobo, ikhadi inethiwekhi, njl. Lspci-isixhobo isetyenziswa ukwenza ulwazi malunga zonke nabaqondisi PCI kwinkqubo, kunye nezixhobo enxulumene kubo.
สำหรับพิมพ์ข้อมูลเกี่ยวกับอุปกรณ์ PCI, คำสั่งต่อไปKjør.
Ukuze ufumane inkcazelo print malunga izixhobo PCI, Kjør umyalo olandelayo.
ใช้ตัวเลือก v ไปยังข้อมูลรายละเอียดเกี่ยวกับการผลิตอุปกรณ์ที่เชื่อมต่อแต่ละ.
Sebenzisa -v khetho ukuba isivuno inkcazelo eneenkcukacha malunga isixhobo ngasinye ezidityanisiweyo.
สำหรับการแสดงทั้งหมด SCSI ของคุณ / อุปกรณ์ SATA, ใช้คำสั่ง lsscsi ดังนี้. หากคุณไม่ได้ติดตั้งเครื่องมือ lsscsi, kjor คำสั่งต่อไปที่จะติดตั้ง.
Kuba show yonke SCSI yakho / izixhobo wada wabiliswa, ukusebenzisa umyalelo lsscsi ngolu hlobo lulandelayo.
