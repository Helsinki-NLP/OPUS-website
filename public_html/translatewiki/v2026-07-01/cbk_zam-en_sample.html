Man zoom por medio de man scroll y man pan por medio de man drag. Cada nodo (= vertex) ta representa un articulo. Ya hace ordena con estos por año estaba arriba (mas nuevo) hasta abajo (mas daan), que cada año tiene otro-otro color. El grandor del maga nodo ta man depende na total del de ila "in-degree", el numero de maga citacion que el maga papel ya recibi estaba na maga input articles, y el de ila 'out-degree', amo el numero de input articules que sila ta hace cita. Hende firme proporcional el local 'in-degree' para na global numero de citacion. El maga nodo tiene corte de redondo amo el maga input articles ( ● ). Maga triangulo que ta enseña para arriba (▲) ta representa el maga sugestion que el vene, que citao el maga input articles pero hende parte del de ila.
Zoom by scrolling and pan by dragging. Each node (= vertex) represents an article. They are ordered by year from top (newer) to bottom (older), with each year colored differently. Alternatively, nodes can also be colored by the article's journal in 'network settings'. The size of the nodes depends on the sum of 'Cited', the number of local incoming citations the paper received from the Seed Articles, and 'Citing', the number of Seed Articles they cite (local outgoing citations). The numbers of local citations are not always proportional to the numbers of global citations. Seed Articles are circle-shaped (●). Up-pointing triangles (▲) represent 'Top Cited', which are cited by the Seed Articles.
Cada borde (= link / arrow) ta representa un referencia y citacion na mismo vez: Maga referencia estos estaba na de ila maga outgoing nodes y maga nodo que el vene. Usualmente, ta puede lang el maga papel cita maga daan papel, por eso, el maga borde firme ta enseña para arriba (o para izquierda o derecha para na maga citacion que del mismo año). Tecnicamente, ta llama cin este como un directed acyclic graph (o directao grafo aciclico).
Each edge (= link / arrow) represents a citation: They are outgoing citations for their source nodes and incoming citations for their target nodes. Usually papers can only cite older papers, hence edges tend to point downwards (or sidewards for citations of the same year). Technically a citation network is a directed acyclic graph.

Este "local" author network ta dale mira con el maga con todo comun autor entre na maga articulo input y sugerido. Depende na disciplina, el numero de maga (co)authors entre na un set de maga cuantos articulo que puede de pronto queda cien-cuen el numero, por eso este lang author network el ta muestra maga autor que tiene minimo numero de maga articulo local (no queda confundido na de ila numero global del maga publicacion, que usualmente alto el numero). Determinao el predeterminao minimo numero de articulo local por eso este network ta dale mira hende mas de 50 maga autor, pero puede lang con este cambia por medio con el slider na abajo desde 2 hasta 10.
This 'local' co-authorship network shows the most common authors among the Seed Articles. Depending on the discipline, the number of (co)authors among a set of multiple articles can quickly rise to the hundreds, which is why this author network only shows authors with a minimum number of Seed Articles (not to be mistaken with their global number of publications, which is usually higher). The default minimum number of Seed Articles is determined so that this network shows no more than 50 authors, but it can be changed with the dropdown at the bottom.
Cada nodo (= vertex) ta representa un autor. Depende el grandor na numero del de ila articulo local y depende el color na author-cluster que donde aparece sila una. Man clic maga autor para hace sala maga articulo sugerido o ya hace input, ctrl+click para hace sala mas de un autor.
Each node (= vertex) represents an author. Their size depends on their number of seed articles and their color depends on the first or last seed article they co-authored. Click authors to filter articles-tables, ctrl+click to filter for more than one author. Drag nodes to rearrange them.
Cada canto (= link) ta representa colaboracion entra medio del dos autor na maga articulo sugerido y ya hace input, y ta indica el anchor con el numero del maga colaboracion. Man clic para hace sala maga articulo sugerido y ya hace input.
Each edge (= link) represents a collaboration between two authors among the seed articles, the width indicating the number of seed articles. Click edges to filter articles-tables.
