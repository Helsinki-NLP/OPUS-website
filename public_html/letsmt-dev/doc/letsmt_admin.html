NAME

letsmt_admin - administrate LetsMT! Resource Repository

USAGE

 letsmt_admin [OPTIONS] COMMAND

COMMANDS list_slots ..... list all slots and branches in the repository list_files ..... list all files (recursively) in a given path check_meta ..... check meta data (find files without metadata records) find_parallel .. find and list parallel documents

OPTIONS -s slot ........ name of the REST-slot (= corpus) -u user ........ name of the REST-user -d path ........ relative path (directory) inside of the repository -f ............. fix problems (repair meta data etc ....) -v ............. verbose output

DESCRIPTION

With this little script you can execute a few administrative actions.
