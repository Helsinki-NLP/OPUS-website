NAME

10_Language_Detect.t - test language-detection facilities

DESCRIPTION

This script tests the following assertions:

  • You can convert latin1-encoded text files to utf8.

  • You can convert Russian KOI8-R-encoded text files to utf8.

  • You can detect various character encodings of texts (English, Swedish, Russian, Chinese; latin1, UTF-8, UTF16, KOI8, Big5).

  • You can detect the language of (utf-8-encoded!) texts (English, Swedish, Russian, Chinese, Croatian).
