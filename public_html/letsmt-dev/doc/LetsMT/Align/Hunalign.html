NAME

LetsMT::Align::Hunalign - Sentence-align with hunalign

DESCRIPTION

Sentence-align two resources using the external tool hunalign.

CONSTRUCTOR

 $aligner = new LetsMT::Align (method => 'hunalign', %params)

METHODS

align

 $AlgRes = align ($SrcRes, $TrgRes [, $AlgRes])

Align $SrcRes and $TrgRes with each other and save sentence alignments in $AlgRes. If no $AlgRes is given: Create a new resource from the information found in $SrcRes, $TrgRes (see LetsMT::Align::make_align_resource).

_hunalign2links

Parse hunalign output and extract sentence links.

_resource2hunalign

Create temporary files for running the aligner.
