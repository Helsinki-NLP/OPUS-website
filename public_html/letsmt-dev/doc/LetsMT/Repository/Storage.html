NAME

LetsMT::Repository::Storage - abstract class for storage in version-control systems

DESCRIPTION

Implemented backends:

 svn_local  (aliases: svn_file, VCSubversion, VCSubersionLocal)
 svn_server (alias:   VCSubversionServer)
 filesystem
 compressed
 any_vc
 vc

CONSTRUCTOR / FACTORY METHOD

 $storage = new LetsMT::Repository::Storage ($type, $resource [, $format])

Return an appropriate reader object for a given resource $resource. The data format $format is optional. The constructor tries to infer the data from the resource object if $format is not specified (see the LetsMT::Resource manpage::type).
