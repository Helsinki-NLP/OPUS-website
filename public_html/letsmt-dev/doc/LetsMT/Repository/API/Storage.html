NAME

LetsMT::Repository::API::Storage - API class for the REST URI storage

SYNOPSIS

 use LetsMT::Repository::API::Storage;
 my $storage = LetsMT::Repository::API::Storage;
 $storage->new($r);
 $storage->process;

DESCRIPTION

This class implements the functionality available through the REST URI storage - GET, PUT, POST and DELETE - and returns the result as a XML string.

get

List all slots or the content of a slot, branch or path, or the content of a file.

put

Upload data and optionally import it.

post

Create or copy a slot, branch, path, or upload data.

delete

Delete a slot, branch or content of a path.

INTERNAL UTILITY METHODS

_get_payload

 $api->_get_payload( $r, $payload_ref )

Receive the payload from PUT or POST and write it to a path.

Has no defined return value.
