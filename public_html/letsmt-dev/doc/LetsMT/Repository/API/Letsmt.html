NAME

LetsMT::Repository::API::Letsmt - API class for the REST URI letsmt

SYNOPSIS

 use LetsMT::Repository::API::Letsmt;
 my $letsmt = LetsMT::Repository::API::Letsmt;
 $letsmt->new($r);
 $letsmt->process;

DESCRIPTION

This class implements the functionality available through the REST URI letsmt - GET, PUT, POST and DELETE - and returns the result as a XML string.

METHODS

get

Execute GET request by forwarding the call to storage API.
