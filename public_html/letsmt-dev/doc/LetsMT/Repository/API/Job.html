NAME

LetsMT::Repository::API::Job - API class for the REST URI job

SYNOPSIS

 use LetsMT::Repository::API::Job;
 my $job = LetsMT::Repository::API::Job;
 $job->new($r);
 $job->process;

DESCRIPTION

This class implements the functionality available through the REST URI job - GET, PUT, POST and DELETE - and returns the result as a XML string.

METHODS

get

Get a job description.

put

Add or overwrite a job description.

post

Submit a new job description.

delete

Delete a running or scheduled job.
