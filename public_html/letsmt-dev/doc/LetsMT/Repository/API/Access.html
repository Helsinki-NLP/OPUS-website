NAME

LetsMT::Repository::API::Group - API class for the REST URI group

SYNOPSIS

 use LetsMT::Repository::API::Access;
 my $access = LetsMT::Repository::API::Access;
 $access->new($r);
 $access->process;

DESCRIPTION

This class implements the functionality available through the REST URI access - GET and PUT - and returns the result as a XML string. Supports showing which group a slot is in and setting a new group for a slot.

METHODS

get

Get the group the given branch/slot is in.

put

Set a new group for the given branch/slot.
