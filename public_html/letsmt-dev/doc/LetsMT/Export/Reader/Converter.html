NAME

LetsMT::Export::Reader::Converter - read documents after converting to XML

DESCRIPTION

A general reader module that first converts to XML (using modules from the Import class) and then reads the converted file.

CONSTRUCTOR

 $reader = new LetsMT::Export::Reader::Converter (format => $format)

Supported formats:

doc
pdf
