NAME

LetsMT::Export::Writer

DESCRIPTION

Resource-writer objects.

Implemented data formats

align - XCES
moses - Moses
para - Parallel corpus
pdf - PDF
srt - SRT
text - Plain text
tmx - TMX
xces - XCES
xliff - XLIFF
xml - XML

METHODS

Constructor

 $writer = new LetsMT::Export::Writer ($resource, $format)

Returns an appropriate writer object for a given resource $resource. The data format $format is optional. The constructor tries to infer the data from the resource object if $format is not specified (see the LetsMT::Resource manpage::type)
