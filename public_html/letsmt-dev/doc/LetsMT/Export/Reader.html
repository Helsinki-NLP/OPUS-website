NAME

LetsMT::Export::Reader - reader objects for various data formats

DESCRIPTION

Reader objects for various data formats.

Implemented data formats

doc - Word DOC
mono - Monolingual corpus
moses - Moses
para - Parallel corpus
pdf - PDF
srt - SRT
text - Plain text
tmx - TMX
xces - XCES
xliff - XLIFF
xml - XML

CONSTRUCTOR / FACTORY METHOD

 $reader = new LetsMT::Export::Reader ($resource, $format)

Return an appropriate reader object for a given resource $resource.

The data format $format is optional. If it is not specified, the constructor tries to infer the data from the resource object (see LetsMT::Resource::type)

CLASS METHOD

reader

 LetsMT::Export::Reader::reader ($format)

Return appropriate reader for a given format.
