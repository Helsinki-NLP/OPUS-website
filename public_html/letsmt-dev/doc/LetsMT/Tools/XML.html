NAME

Tools

DESCRIPTION

XML-related tools

FUNCTIONS

validate_dtd

 $errors = &validate_dtd ($resource, $dtd)

Validate $resource according to $dtd.

Returns a list of errors if it fails, and an empty list otherwise.

validate_xs

 $errors = &validate_xs ($resource, $xs)

Validate $resource according to XML Schema $xs. Returns a list of errors if it fails, and an empty list otherwise.

validate_xml

 $errors = &validate_xml ($resource)

Check that $resource is valid XML. This expects a DOCTYPE declaration in the document!

Returns a list of errors if it fails, and an empty list otherwise.

validate_standalone_xml

 $errors = &validate_standalone_xml ($resource)

Check that $resource is valid xml (without DTD validation id a DOCTYPE is given in the document).

Returns a list of errors if it fails, and an empty list otherwise.

tidy_xml

 $errors = &tidy_xml ($resource)

Validate, correct, and format (in place) the XML file found at path $resource.

Returns a list of errors if it fails, and an empty list otherwise.

clean_xml

Clean up strings before parsing.
