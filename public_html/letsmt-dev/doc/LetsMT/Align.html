NAME

LetsMT::Align - family of modules for sentence alignment

DESCRIPTION

A factory class to return an object instance of a selected alignment module.

CONSTRUCTOR

 $aligner = new LetsMT::Align (method => 'hunalign|gale|one-to-one|bisent', %params)

Return an object instance of the alignment module that is selected with the 'method' argument. Available methods:

  • Hunalign
  • One-to-one
  • Gale-Church
  • Hunalign in 'bisent' mode

Default is the Hunalign sentence aligner.

%params is a hash of additional parameters (specific to the alignment method chosen).

METHODS

align_resources

 $AlignRes = $aligner->align_resources($SrcRes, $TrgRes [, $AlignRes])

Align sentences of two given resources ($SrcRes,$TrgRes) in the resource repository. Returns the resource object ($AlgRes) that stores the sentence alignment.

The $AlignRes argument is optional. If it is not given, a clone of the $SrcRes is used with the language part of its path replaced by the language pair (language of $SrcRes + language of $TrgRes, sorted alphabetically) (see make_align_resource).

make_align_resource

 $AlignRes = $aligner->make_align_resource($SrcRes, $TrgRes)

Create a resource for storing sentence alignment between $SrcRes and $TrgRes.

$AlignRes will be a clone of $SrcRes except for the language part of the path. The language will be replaced by the language pair taken from the given resources $SrcRes and $TrgRes.
