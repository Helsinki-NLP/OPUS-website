NAME

LetsMT::Import::XLIFFReader - reader for XLIFF files

DESCRIPTION

XLIFFF: the XML Localization Interchange File Format.

CLASS VARIABLE (internal)

Tags

Tags can be either extracted or ignored. The class variable $TAG_TO_CONCEPT maps tmx-tags to keys that the reader can use. If the reader hans any such key set to a true value, the tags will be visible. They are thus ignoed by default. All tags mapped for TMX are also mapped for XLIFF .

read

Read one translation unit from the open xliff-file.

Returns a reference to a translation unit or undef. A translation unit always has the format:

 {
     lang1 => {
         id1 => [ token1, token2, ..., tokenN ],
         id2 => [ token1, token2, ..., tokenN ],
         ...
     },
     lang2 => {
         id1 => [ token1, token2, ..., tokenN ],
         id2 => [ token1, token2, ..., tokenN ],
         ...
     },
     ...
 }
