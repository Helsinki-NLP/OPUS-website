NAME

LetsMT::Import::OPUS - import handler for OPUS corpus data

DESCRIPTION

OPUS: an open parallel corpus.

A child of LetsMT::Corpus.

CONSTRUCTOR

 $opus = new LetsMT::Import::OPUS (%OPTIONS)

OPTIONS:

 -rest_home (default: LETSMT_URL)
 -opus_home (default: '/home/opus/OPUS/corpus/')
 ...

METHODS

import_corpus

 $opus->import_corpus ($corpus, $destination, $user)

Import corpus from opus_home/$corpus into LetsMT/$destination/$user.
