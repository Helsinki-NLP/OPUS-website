NAME

LetsMT::Import::TextReader - reader for plain-text files

CONSTRUCTOR

 $reader = new LetsMT::Import::TextReader (%OPTIONS)

Create a new instance of LetsMT::Import::TextReader, which is guaranteed to have the fields tokenizer, normalizer, splitter and lang. If no such keys are provided in the supplied %hash, the defaults in LetsMT::Import are used.

All sentences begin read are returned as if they are in language lang. This has consequences for the way they are written with LetsMT::Import::XCESWriter.

METHODS

open

read

 $reader->read
 $reader->read ($before, $after)

... optional hash structures for data before and after the current text chunk (par-breaks).

Monolingual sentence "pair" structure:

 {
     lang => {
         id => [ token1, token2, ..., tokenN ]
     }
 }

Before and after markup structures:

 {
     lang => [
         "plain text",
         [open-tag, {attr1 => 1, ... } ],
         [close-tag],
     ]
 }
