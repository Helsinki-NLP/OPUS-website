NAME

LetsMT::Import::MosesReader - reader for Moses files

DESCRIPTION

Moses: plain-text data used for statistical machine translation systems.

CLASS VARIABLE (public)

$DEFAULT_NORMALIZER

An instance of the whitespace normalizer.

CONSTRUCTOR

 $reader = new LetsMT::Import::TextReader (%OPTIONS)

Creates a new instance of LetsMT::Import::TextReader, which is guaranteed to have the fields tokenizer, normalizer, splitter and lang. For any key not provided in the supplied %OPTIONS, the defaults in LetsMT::Import are used.

All sentences begin read are returned as if they are in language lang. This has consequences for the way they are written with LetsMT::Import::XCESWriter.

METHODS
