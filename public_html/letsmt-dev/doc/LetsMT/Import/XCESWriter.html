NAME

LetsMT::Import::XCESWriter - Write files in the internal XCES format

CLASS VARIABLE (public)

$DEFAULT_NORMALIZER

Default text normalizer to be applied before writing sentences to XML. This will remove initial/trailing/duplicated whitespaces and also all special characters that may break Moses.

CONSTRUCTOR

METHODS

open

 $writer->open ($resource, $encoding)

Open a resource for writing. If $encoding is left undefined, whatever is defined as the default writing encoding in LetsMT::Tools will be used.

write

 $writer->write ($tu)

Write translation unit $tu to the open resource.

get_xml_writer

 $writer->get_xml_writer ($lang)

Return language-specific XML writer object (create if necessary).

get_xces_writer

 $writer->get_xces_writer ($src_lang, $trg_lang)

Return language-pair-specific XCES alignment writer object (create if necessary).

close

get_resources

Returns a reference to the list of resources that were writte to. The return value is formatted to be useful to LetsMT::Import.
