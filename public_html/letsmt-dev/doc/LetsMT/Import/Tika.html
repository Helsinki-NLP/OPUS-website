NAME

LetsMT::Import::Tika - generic import handler for documents handled by Tika

DESCRIPTION

This module uses the Apache tool Tika to validate and extract text from various documents.

CONSTRUCTOR

METHODS

validate

 $handler->validate ($resource)

Validates that $resource is a doc file.

convert

 $handler->convert ($resource, $importer)

Convert $resource to text and import the resulting text file.
