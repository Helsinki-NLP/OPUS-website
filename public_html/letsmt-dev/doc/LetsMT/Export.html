NAME

LetsMT::Export - family of modules for exporting data

DESCRIPTION

Export resources from the repository into various formats.

This parent class is rarely called directly.

CONSTRUCTOR

 $exporter = new LetsMT::Export ( -verbose => 1, local_root => '...' )

Construct a new export object.

CLASS METHODS

Filter settings

With two input parameters $filter and $value, set a filter property. With one input $filter, read the filter's respective setting.

max

 $exporter->max ($filter, $count)
 $count = $exporter->max ($filter)

Read at most $count data records.

skip

 $skips = $exporter->skip ($filter)
 $exporter->skip ($filter, $skips)

Skip $skips data records.

link_type

 $exporter->link_type ($filter, $type)
 $type = $exporter->link_type ($filter)

Return only sentence alignments of a certain type $filter.

The link_type filter is only useful for parallel corpora. The link type ($type) is a simple string, for example, '1:1' to require one-to-one links.

reset

 $exporter->reset ($filter)

Reset exporter filters and counters.

INSTANCE METHODS

export_resource

 $exporter->export_resource($FromResource,$ToResource,$FromFormat,$ToFormat)

Reads data from $FromResource and writes them to $ToResource. $FromFormat and $ToFormat are optional and may be infered from the path of the given resources (see the LetsMT::Resource manpage::type).

$FromResource, $ToResource .. LetsMTResource objects

$FromType, $ToType .. strings

See the LetsMT::Export::Reader manpage and the LetsMT::Export::Writer manpage for supported resource formats.

Returns a list of resources (LetsMT::Resource objects) created while exporting.
