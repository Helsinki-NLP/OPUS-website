NAME

LetsMT::DataProcessing::Normalizer

DESCRIPTION

Derivative classes must implement normalize_no_copy.

CONSTRUCTOR

 $normalizer = new LetsMT::DataProcessing::Normalizer ( type => $types )

$types can be a comma-separated list of normalizer types.

METHODS

normalize_no_copy

 $normalizer->normalize_no_copy ($string)

Normalize $string destructively (the original string passed as parameter is changed).

Abstract method, must be overridden! It is also used by the inherited copying method normalize.

normalize

 $normalized_string = $normalizer->normalize ($string)

Normalize a copy of $string and return it.

Use normalize_no_copy instead if the original string is not needed.
