NAME

LetsMT::DataProcessing::Splitter

DESCRIPTION

Split texts into sentences.

Derivative classes should implement split.

CONSTRUCTOR

 $splitter = new LetsMT::DataProcessing::Splitter (%OPTIONS)

OPTIONS:

 method ........... (possible values: europarl, lingua, or opennpl;
                     default: no splitting)
 lang ............. Language code (default/fallback: en)
 separate_lines ... Keep separate lines in input? (default: false)

METHODS

split

 ( s_1, s_2, ..., s_n ) = $splitter->split (f_1, f_2, ..., f_m)

where s_i is an identified sentence and f_i is a text fragment.

Turn a list of text fragments into a list of sentences (where it is not certain that the last one is a proper sentence, since there might be a missing fragment).

Uses consecutive calls passing the last sentence of one call as the first fragment of the next call.

Abstract method, must be overridden!
